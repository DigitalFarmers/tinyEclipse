== Description ==

Adds the TinyEclipse AI chat widget to your WordPress site.
Each site gets its own isolated AI brain that learns from your content.

== Installation ==

1. Upload the `tinyeclipse-widget` folder to `/wp-content/plugins/`
2. Activate the plugin in WordPress → Plugins
3. Go to Settings → TinyEclipse
4. Enter your Tenant ID (get it from the TinyEclipse admin panel)
5. Done — the AI widget appears on your site
